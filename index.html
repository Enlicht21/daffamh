    Lines Detection using OpenCV

    In 2019 during my first year at the Brawijaya Robotics Team organization, our team had a problem with how a Quadruped Robot should be able to run autonomously according to the rules of the competition. Therefore, I had an idea and took the initiative to do a project using a camera to detect the available paths in the arena to guide a Quadruped Robot movement.     Optimization Project

    This is my final project, the purpose of this project is to determine the DC motor parameters by using the Particle Swarm Optimization (PSO) algorithm. In this study, the fitness function of the optimization problem is used to determine the performance of the algorithm used by the Mean Squared Error (MSE) method and the PSO algorithm is processed by the Arduino Uno microcontroller. Input from the system is the angular velocity value obtained from the incremental rotary encoder sensor and the current passing through the motor obtained from the current sensor.

    This project is documented in the form of a student journal which can be found here.
